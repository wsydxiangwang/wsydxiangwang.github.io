# 实现一个无限累加函数

问题：用 JS 实现一个无限累加的函数 add，示例如下：

add(1); // 1
add(1)(2);  // 3
add(1)(2)(3)； // 6
add(1)(2)(3)(4)； // 10 

// 以此类推

实现：

function add(a) {
	function sum(b) { // 使用闭包
    	a = a + b; // 累加
    	return sum;
 	}
 	sum.toString = function() { // 重写toString()方法
        return a;
    }
 	return sum; // 返回一个函数
}

add(1); // 1
add(1)(2);  // 3
add(1)(2)(3)； // 6
add(1)(2)(3)(4)； // 10

我们知道打印函数时会自动调用toString()方法，函数add(a)返回一个闭包sum(b)，函数sum()中累加计算 a = a + b，只需要重写sum.toString()方法返回变量 a 就OK了。
